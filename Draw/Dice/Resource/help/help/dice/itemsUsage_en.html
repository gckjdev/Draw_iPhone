By using "CUT" item, you means you are quite confident to challenge, so the ante doubles.
If you are not content with the rolled dice, you can use "Reroll" item to re-roll once.A round can only be used once.
By using "Peek" item, you can peek one dice of each player.You can use unlimited number of times.
By using "Postpone" item, you can get more time before you should make a bid or challenge. You can use it once in one game.
Contrast to "Postpone" item, "Urge" item will cut off 5 seconds from next player.A round can only be used once.
Like pass in Pokers, you can use "Turtle" item to yield your chance if you feel not safe to bid or challenge.A round only be used once, three or more players can use the item.
By using "Reverse" item, you can reverse the play order,which is clockwise by default.You can use unlimited number of times.
By using "Smart butler" item, the game system will give your advice of whether challenge or bidding, and what to bid.You can use unlimited number of times.
